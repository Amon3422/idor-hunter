How to get a cURL command

            
    
              
  1. Open Chrome DevTools (F12)
    2. 
              
  2. Go to the Network tab
    3. 
              
  3. Perform the action you want to test
    4. 
              
  4. Right-click the request → Copy → Copy as cURL
    5. 
              
  5. Paste it in the textarea above
    6. 
            

          

          

            
What happens next

            
    
              
  1. The cURL is parsed to extract URL, method, headers, and body
    2. 
              
  2. These become Account A (the victim) credentials
    3. 
              
  3. You provide Account B (attacker) credentials
    4. 
              
  4. Both requests are replayed and the responses are diffed
    5. 
              
  5. Suspicious diffs are analyzed by the AI engine
